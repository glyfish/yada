name: Double spacing between functions
description: Always use two blank lines between top-level function and class definitions in Python files
type: feedback
---

Use two blank lines between top-level function and class definitions in Python files.

**Why:** User preference for consistent PEP 8 style spacing.

**How to apply:** Any time writing or editing Python files, ensure two blank lines separate top-level definitions (functions, classes). Applies project-wide going forward.
